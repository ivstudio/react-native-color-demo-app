# React Native Color App

Multi-screen React Native demo App incorporating Navigation, AsyncStore, Fetching Data, ScrollView, ListView and more.

### Libraries Used

- [React Native Docs](https://facebook.github.io/react-native/docs/getting-started.html)
- [Color](https://www.npmjs.com/package/color)


To run the app:

 ```sh
 npm install
 react-native run-ios
 ```
